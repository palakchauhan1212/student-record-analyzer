Word Frequency Analyzer is a beginner-friendly Python project that analyzes a paragraph entered by the user. The program converts the text to lowercase, removes basic punctuation, calculates the frequency of each word, identifies the most frequently occurring word, and displays all unique words.

Features

- Accept a paragraph as input
- Convert all text to lowercase
- Remove basic punctuation marks
- Count the frequency of each word
- Identify the most frequently occurring word
- Display all unique words

Concepts Used

- Classes and Objects
- Dictionaries
- Sets
- Strings
- Loops
- Conditional Statements
- String Methods ("lower()" and "split()")
- User Input

Requirements

- Python 3.x

How to Run

1. Download or clone this repository.
2. Open a terminal in the project folder.
3. Run the following command:

python word_frequency_analyzer.py

«Replace "word_frequency_analyzer.py" with your actual Python filename if it has a different name.»

Sample Output

The program displays:

- The processed paragraph (lowercase and without basic punctuation)
- Frequency of each word
- Most repeated word
- Set of unique words
