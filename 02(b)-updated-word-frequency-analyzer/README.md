Word Frequency Analyzer is a beginner-friendly Python project that analyzes a paragraph entered by the user. The program converts the text to lowercase, removes basic punctuation, counts the frequency of each word, identifies the most frequently occurring word, and displays all unique words.

Features

- Accept a paragraph as input
- Convert text to lowercase
- Remove basic punctuation characters
- Count the frequency of each word
- Find the most repeated word
- Display all unique words

Concepts Used

- Functions
- Dictionaries
- Sets
- Strings
- Loops
- Conditional Statements
- User Input
- Lambda Functions
- String Methods ("lower()" and "split()")

Requirements

- Python 3.x

How to Run

1. Download or clone this repository.
2. Open a terminal in the project folder.
3. Run the following command:

python word_frequency_analyzer.py

«Replace "word_frequency_analyzer.py" with your actual Python filename if it has a different name.»

Sample Output

The program displays:

- Frequency of each word
- Most repeated word and its occurrence count
- Set of unique words
