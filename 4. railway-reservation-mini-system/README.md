Railway Reservation System is a beginner-friendly Python project that simulates a basic railway ticket reservation process. The system allows users to book tickets, cancel reservations, rebook cancelled seats, and view passenger details through a menu-driven interface.

Features

- Book one or more railway tickets
- Automatically assign seat numbers to passengers
- Prevent duplicate passenger bookings
- Cancel booked tickets
- Rebook available cancelled seats
- Display the list of booked passengers and their seat numbers
- Display available (cancelled) seat numbers

Concepts Used

- Classes and Objects
- Dictionaries
- Lists
- Loops
- Conditional Statements
- Functions
- Encapsulation (Private Attributes)
- User Input
- Menu-Driven Programming

Requirements

- Python 3.x

How to Run

1. Download or clone this repository.
2. Open a terminal in the project folder.
3. Run the following command:

python railway_reservation_system.py

«Replace "railway_reservation_system.py" with your actual Python filename if it has a different name.»

Sample Operations

The program allows users to:

- Book tickets
- Cancel booked tickets
- Rebook cancelled seats
- Display passenger details
- Display available seat numbers
- Exit the application
