Password Strength Checker

Description

Password Strength Checker is a beginner-friendly Python project that evaluates the strength of a user-entered password based on common password security rules. The program checks whether the password meets the required criteria and displays helpful feedback for any missing requirements.

Features

- Accept a password from the user
- Check for a minimum password length of 8 characters
- Verify the presence of at least one uppercase letter
- Verify the presence of at least one lowercase letter
- Verify the presence of at least one digit
- Verify the presence of at least one special character
- Display whether the password is Strong or Weak
- Show all missing password requirements

Concepts Used

- Functions
- Lists
- Loops
- Conditional Statements
- String Methods ("isupper()", "islower()", "isdigit()", "isalnum()")
- User Input

Requirements

- Python 3.x

How to Run

1. Download or clone this repository.
2. Open a terminal in the project folder.
3. Run the following command:

python password_strength_checker.py

«Replace "password_strength_checker.py" with your actual Python filename if it has a different name.»

Sample Output

The program displays:

- Password Strength (Strong or Weak)
- Missing password requirements, if any
