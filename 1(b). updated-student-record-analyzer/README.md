Student Record Analyzer is a beginner-friendly Python project that collects student names and marks, then analyzes the data to generate useful insights about the class.

The program calculates the average marks, identifies the class topper, displays the names of students who scored below the passing marks, and shows all unique marks obtained by the students.

Features

- Add records for multiple students
- Store student names and marks using a dictionary
- Calculate the average marks of the class
- Identify the class topper
- Display students who scored below 40 marks
- Display all unique marks in the class

Concepts Used

- Functions
- Dictionaries
- Lists
- Sets
- List Comprehensions
- Lambda Functions
- Built-in Functions ("sum()" and "max()")
- User Input

Requirements

- Python 3.x

How to Run

1. Download or clone this repository.
2. Open a terminal in the project folder.
3. Run the following command:

python student_manager.py

«Replace "student_manager.py" with your actual Python filename if it has a different name.»

Sample Output

The program displays:

- Average Marks
- Class Topper
- Failed Students
- Unique Marks
